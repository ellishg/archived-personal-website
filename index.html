Fluid Simulation

In computer graphics, fluid is used to describe systems like water, smoke, and dust. It is notoriously hard to simulate in real-time, so particle systems are usually used to approximate them. However, non-particle based fluid simulations can make fluid effects more realistic. I have decided to create a real-time fluid simulation using fast algorithms based on the Navier-Stokes equations that run on the GPU using OpenCL. From this project I have learned a lot about fluid dynamics, how to make a fast implimentation of vector field something something, the architecture of the GPU, how to manage memory efficiently with OpenCL, and interoperability between OpenGL and OpenCL.
